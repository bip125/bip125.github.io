About

I write tools that solve problems I actually have. FFmpeg's bitrate stats weren't detailed enough, so I wrote my own analyzer in C. Running code in 16 languages meant juggling 16 commands, so I built a universal runner in shell. Broadcom Wi-Fi didn't work offline, so I scripted the entire driver install.

Mostly C, Rust, and shell. No frameworks where a Makefile will do. This site is plain HTML assembled with sed — no bundler, no runtime, no dependencies beyond a POSIX shell.

If it ships, it works. No half-finished prototypes.
